# Warfront

Многопользовательская игра захвата территорий в духе [openfront.io](https://openfront.io) / territorial.io.
React 19 + Canvas на клиенте, Node.js + WebSocket (авторитарный сервер) — вся игровая логика на сервере.

## Запуск

```bash
npm install
npm run dev        # сервер :8080 + vite :5173
```

Открыть http://localhost:5173 (можно в нескольких вкладках — это и есть онлайн).

Продакшен-режим (один порт, сервер раздаёт статику):

```bash
npm run build
npm start          # http://localhost:8080
```

## Как протестировать онлайн

- **Один комп**: открыть несколько вкладок — каждая вкладка отдельный игрок.
- **Своя сеть (телефон, второй ноут)**: `npm run build && npm start`, узнать IP
  (`ipconfig getifaddr en0`) и открыть `http://<IP>:8080` с другого устройства.
- **Через интернет (друзья)**: `npx cloudflared tunnel --url http://localhost:8080` —
  даст публичный https-адрес, WebSocket через него работает (клиент сам
  переключается на wss при https).

## Как играть

- **Режимы**: быстрая игра (общая комната), создать лобби (код из 5 символов для
  друзей, хост выбирает карту и сложность), войти по коду. До 100 человек в комнате.
- **Карты**: случайный мир (новые континенты каждый раунд) или Земля
  (реальные материки, Антарктида и крупные острова, 480×240).
- Быстрая игра замораживается без людей и сбрасывается, когда выходит последний
  человек. Высадка возможна даже на занятую ботами землю — плацдарм вырезается
  из их территории (рядом с людьми высадиться нельзя).
- После старта — 20 секунд на выбор точки старта кликом; когда все выбрали,
  игра стартует досрочно, опоздавших спавнит случайно.
- Клик по нейтральной или вражеской территории рядом со своей — атака.
  Ползунок внизу — какой процент войск отправить.
- Захват волновой: граница движется сплошным фронтом, без дыр. Встречные
  атаки взаимно уничтожаются 1:1 — граница идёт в сторону того, кто выделил
  больше войск. Линия фронта подсвечивается красным, у неё видно выделенные
  войска (⚔️ свои / 🛡 противника).
- Прирост войск процентный (от текущей армии), потолок зависит от территории.
- Карта 400×400. Боты двух видов: слабые со случайными именами (🤖) и сильные
  «страны» с флагами (🇷🇺 Россия, 🇺🇸 США…) — растут заметно быстрее.
  Сложность: лёгкая 50 ботов (5 стран), средняя 70 (15), сложная 90 (40).
- Колесо — зум, перетаскивание — панорама, мини-карта слева внизу.
- Побеждает тот, кто захватит 65% суши. Потом автоматический новый раунд.

## Архитектура

```
shared/protocol.ts   — типы сообщений и константы (карта 200×200, тик 100 мс)
shared/color.ts      — детерминированный цвет игрока по id
server/game.ts       — игровая логика: генерация карты, рост войск, атаки, боты
server/index.ts      — HTTP + WebSocket, игровой цикл, рассылка диффов
client/game-client.ts— рендер (ImageData → canvas), пан/зум, клики
client/App.tsx       — UI: меню, HUD, лидерборд
```

Синхронизация: при подключении клиент получает полный снапшот карты, дальше —
только диффы изменённых клеток каждый тик (`[cell, owner, ...]`).
